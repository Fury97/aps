Second Normal Form     第二范式     关系中所有的非主属性(Prime Attribute)不存在对主属性的部分函数依赖。换句话说，所有非键属性都 完全函数依赖于 R的候选键。When all None key attribute fully functional depend on key attribute, it satisfies second normal form. Etc, student id, class id, age, class grade make up a table, where (student id, class id) can be chosen as key, but age partially functionally depend on student id. So this dosen’t satisfy second normal form.

BCNF, Boyce-Cott Normal Form      所有的函数依赖都是 键->属性 的。

3NF, third Normal Form     第三范式     不存在任何非主属性对键的传递依赖。反过来说，所有的属性都是从主键中获取出来的。When a relation R satisfies 2NF, and there exists no transit dependency in R, then R satisfies 3NF. 处理第三范式需要将传递的部分拆开来。
